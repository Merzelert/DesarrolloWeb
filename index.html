Etiquetas semanticas estructurales

div - Es una etiqueta contenedora que no tiene una semantica
header - cabecera de un sitio web o de una seccion
main - Define la seccion principal del documento, solo puede existir una etiqueta main por documento.
footer - Pie de pagina de un sitio web o de una seccion
nav - Representa una navegacion
article - Representa una seccion de autocontenido (que por si solo se explica)
aside - Representa ucontenido complementario o secundario
section - Representa una seccion de contenido generico
addres- Representa una direccion o informacion de contacto

Etiquetas de bloque vs Etiquetas de linea

La etiqueta de bloque por escelencia es la div, una etiqueta de bloque es aquella que generta saltos de linea con sus etiquetas hermanas
La etique de linea por excelencia es la span, una etiqueta de linea solo ocupa el espacio necesario que tiene su contenido Hola soy otra span
